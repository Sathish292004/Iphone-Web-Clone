# 📖 Overview

iPhone Web Clone is a visually stunning recreation of Apple's iPhone 15 Pro product page. The project focuses on delivering a premium user experience through smooth animations, interactive 3D product visualization, responsive layouts, and modern frontend development practices.

---

# ✨ Features

* 📱 Apple-Inspired Landing Page
* 🎥 Interactive Video Showcase
* 🧊 3D iPhone Model Viewer
* 🎨 Dynamic Color Selection
* ⚡ GSAP Animations
* 🎯 Smooth Scroll Effects
* 📺 Product Highlights Section
* 🔄 Video Carousel
* 🌐 Responsive Design
* 🚀 Fast Performance with Vite

---

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sathish292004/Iphone-Web-Clone.git
```

### Navigate to Project

```bash
cd Iphone-Web-Clone
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```
